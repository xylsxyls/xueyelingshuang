void fun(int x, int&y, int z)
{
	printf("&x = %x, &y = %x, *y = %x, &z = %x", &x, &x + 1, *(&x + 1), &z);
	return;
}

int a = 1;
	int b = 2;
	int c = 3;
	printf("&a = %x, &b = %x, &c = %x\n", &a, &b, &c);
	fun(a, b, c);

可以看到y也是占用4字节的空间，内部存放的值就是b的地址
这个依赖于编译器的实现，有的可能不在栈空间里，可能在常量区中
内存分为堆栈数据段和代码段
数据段包含字符串常量全局和静态变量
代码段包含代码中的字符
堆和自由存储区的区别，堆一般指内存分配，就是malloc，自由存储指对象分配，但是本质上分配的区域都是同一块，都属于堆，堆是更大的概念
